# Amazon Sales Data Analysis

## Project Objective
Analyze Amazon sales data to find total sales, best-selling products, order status, city-wise sales, and monthly trends.

## Tools Used
- Excel
- Power BI
- GitHub

## Steps Performed
1. Cleaned the data
2. Changed price format to number/currency
3. Removed duplicates
4. Checked missing values
5. Created pivot tables
6. Built dashboard charts

## Key Insights
- Total sales amount
- Top selling categories/products
- Highest sales city/state
- Monthly sales trend
- Cancelled and shipped order count
